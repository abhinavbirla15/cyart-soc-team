# cyart-soc-team
SOC Operations – Alert Management and Incident Response
## 📌 Overview

This repository contains the work completed for **Task 07 of the CyArt SOC Internship**.  
The focus of this week was to understand and perform **Security Operations Center (SOC)** tasks including alert management, incident response, and threat analysis.

---

## 🎯 Objectives

- Understand alert prioritization in SOC  
- Classify security incidents using standard frameworks  
- Perform alert triage and validation  
- Learn incident response lifecycle  
- Practice evidence preservation  
- Simulate a real-world attack and response  

---

## 🛠 Tools Used

- Google Sheets → Alert management  
- Wireshark → Network analysis  
- tcpdump → Packet capture  
- VirusTotal → Threat intelligence  
- AlienVault OTX → IOC validation  
- Metasploit → Attack simulation  
- Linux Terminal → Hash generation  

---

## ⚙️ Workflow

### 1. Alert Management
- Created alert classification table  
- Assigned priority levels based on severity  
- Mapped alerts to MITRE ATT&CK  

---

### 2. Alert Triage
- Analyzed alerts (e.g., brute-force attempts)  
- Checked source IPs  
- Validated using threat intelligence tools  

---

### 3. Incident Response
- Followed incident response lifecycle:
  - Identification  
  - Containment  
  - Eradication  
  - Recovery  

---

### 4. Evidence Preservation
- Collected sample evidence  
- Generated SHA256 hash  
- Documented integrity  

---

### 5. Capstone Project
- Simulated attack using Metasploit  
- Detected suspicious activity  
- Performed triage  
- Responded by blocking attacker  
- Documented full workflow
